# 📝 To-Do List App

A simple and user-friendly To-Do List application built with the MERN stack (MongoDB, Express, React, Node.js).

## 🚀 Features

- ✅ Add tasks
- ✏️ Edit existing tasks
- 🗑️ Delete tasks
- 📌 Mark tasks as completed
- 🌙 Dark mode (optional)
- 📦 Full MERN stack integration

## 🔧 Technologies Used

- Frontend: React
- Backend: Node.js + Express
